# Form_validation_in_PHP
simple form validation using PHP,HTML and CSS

## validation using reguler expression 

```php
    // for name validation
   (!preg_match("/^[A-Za-z. ]*$/",$name))
   
   // for email validation
   (!preg_match("/[A-za-z0-9._ ]{3,}@[A-za-z0-9._ ]{3,}[.]{1}[A-za-z0-9._ ]{3,}/"))
   
```
